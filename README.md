# Onion Coin Website

A modern, high-performance landing page for Onion Coin featuring advanced animations, smooth scrolling, and a premium user experience.

## Features

- **Preloader**: Custom preloader with progress bar animation.
- **Smooth Scrolling**: Powered by [Lenis](https://github.com/studio-freight/lenis).
- **GSAP Animations**: Advanced animations using [GSAP](https://greensock.com/) and [ScrollTrigger](https://greensock.com/docs/Plugins/ScrollTrigger).
- **SplitType**: Text animations using [SplitType](https://github.com/codrops/SplitType/).
- **Custom Cursor**: Interactive cursor with hover effects.
- **Parallax Effects**: Scroll-triggered parallax for cards and elements.
- **3D Coin**: Interactive 3D spinning coin animation.
- **Glassmorphism**: Modern glassmorphism design with neon accents.
- **Responsive Design**: Optimized for both desktop and mobile devices.

## Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    cd Onion_coin_intro_page
    ```

2.  Open `index.html` in your web browser.

## Dependencies

The project uses the following libraries:

- [Lenis](https://github.com/studio-freight/lenis) - Smooth scrolling
- [GSAP](https://greensock.com/) - Animation library
- [SplitType](https://github.com/codrops/SplitType/) - Text splitting for animations
